Module lua5x-operations

lua5x-operations

Table with simple operations that are not compatible between versions 5.1, 5.2 and 5.3 this is done by `require ('lua5x-operations.' .. _VERSION:sub(-3):gsub("(%.)", "_"))`

Usage:

    local operations = "lua5x-operations"

local a = 1 -- 0b0001
