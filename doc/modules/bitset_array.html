Module bitset_array

bitset array library

Functions

copy (bitset_array) copies a bitset_array in a new one.
equals (bitset_l, bitset_r) return if bitset_array values are equal
band (bitset_l, bitset_r) creates a new bitset_array with the result of the & bitwise operation between two bitset_arrays
bor (bitset_l, bitset_r) creates a new bitset_array with the result of the | bitwise operation between two bitset_arrays
bxor (bitset_l, bitset_r) creates a new bitset_array with the result of the (binary) ~ bitwise operation between two bitset_arrays
bxor (bitset_array) creates a new bitset_array with the result of the (unary) ~ bitwise operation in a bitset_array
lshift (bitset_array, steps) creates a new bitset_array with the result of the << bitwise operation between two bitset_arrays if a bit overflows, it will go to the next index
rshift (bitset_array, steps) creates a new bitset_array with the result of the >> bitwise operation between two bitset_arrays if a bit overflows, it will go to the previous index
