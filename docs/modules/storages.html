new_entry (storage, new_entry_content)
create a new entry in storage, uses a free id and entry space if possible

Parameters:

  • storage storage storage to add the new entry
  • new_entry_content any value, the content of this entry

Returns:

    id id of the new entry

See also:

    entry

Usage:

    -- require 'storages'
local storages = require 'motor.storages'

-- create a new storage
local my_storage_id = storages.new_storage()

-- create a new entry in this storage, with content "10"
local my_entry_id = storages.new_entry(my_storage_id, 10)

delete_dead_entries (storage)
Deletes dead entries , killed by kill_entry, basically executes delete_entry in dead entries, returns if any entry has been deleted.

Parameters:

  • storage storage storage to delete dead entries

Returns:

    boolean true if any entry was deleted

See also:

    kill_entry
